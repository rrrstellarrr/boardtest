
# SpringBoot 게시판 미니프로젝트


### 1. 사용 기술 및 개발 환경
  * Language :
    * JAVA
  * Framework :
    *  Spring-Boot 2.6.7
  * Library :
    *  JPA(Spring Data JPA)
    *  Thymeleaf
    *  Spring Security
  * Build Tool :
    *  Gradle 7.2
  * DataBase :
    *  MySql

### 2. 기능
  * 게시판
    *  게시글 등록
    *  게시글 조회
    *  게시글 수정
    *  게시글 삭제
    *  게시글 페이징 및 검색

  * 사용자
    * 회원 가입
      *  회원가입시 유효성 검사 및 중복 검사
    *  회원 로그인
    *  회원 정보 수정
      *  프로필 사진 등록
      *  회원정보 수정시 유효성 검사 및 중복 검사
    *  회원 탈퇴
      *  회원 탈퇴시 작성된 댓글, 게시글 삭제
  
  * 댓글
    *  댓글 작성
    *  댓글 삭제
